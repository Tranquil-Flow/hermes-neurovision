# Hermes Vision v0.1.1 - Implementation Complete ✅

## Summary

Successfully implemented the theme export/import system for v0.1.1. Users can now design themes with AI agents and share them as portable .hvtheme files.

## Features Implemented

### 1. Theme Export System

**File**: `hermes_vision/export.py`

- Exports themes to `.hvtheme` JSON format
- Includes metadata (name, author, description, timestamp)
- Includes full ThemeConfig parameters
- Base64 encodes custom plugin code
- Converts curses color constants to strings

**CLI Usage**:
```bash
# Export theme
hermes-vision --export neural-sky

# Export with custom metadata
hermes-vision --export neural-sky --output my-theme.hvtheme --author "MyName"
```

### 2. Theme Import System

**File**: `hermes_vision/import_theme.py`

- Imports .hvtheme files with version checking
- Preview mode to inspect before installing
- Security warnings for custom plugins
- User confirmation for custom code
- Automatic migration from older versions
- Registry tracking for all imported themes

**CLI Usage**:
```bash
# Preview theme
hermes-vision --import theme.hvtheme --preview

# Import theme
hermes-vision --import theme.hvtheme

# Import and trust (skip confirmation)
hermes-vision --import theme.hvtheme --trust

# List imported themes
hermes-vision --list-themes
```

### 3. Version Compatibility

**File**: `VERSION_COMPATIBILITY.md`

- Format version checking (1.0, 1.1, 2.0, etc.)
- Graceful handling of version mismatches
- Clear error messages for incompatible versions
- Automatic migration from pre-release formats
- Warnings for newer minor versions
- Hard rejection of newer major versions

**Compatibility Matrix**:
- v1.0 (current) ✓ Full support
- v1.1 (future minor) ⚠ Warning, import with defaults
- v0.9 (pre-release) ⚠ Auto-migrate
- v2.0 (future major) ✗ Error, upgrade required

### 4. Runtime Theme Registration

**Modified**: `hermes_vision/themes.py`

- `_runtime_configs` dict for imported theme configs
- `build_theme_config()` checks runtime registry first
- Allows imported themes to work without code installation

**Modified**: `hermes_vision/theme_plugins/__init__.py`

- `_runtime_plugins` dict for imported theme plugins
- `get_plugin()` checks runtime registry first
- Enables custom plugins from imported themes

### 5. Theme Registry

**File**: `~/.hermes/vision/theme_registry.json`

- Tracks all imported themes
- Metadata: name, title, author, description
- Installation timestamp
- Source file path
- Whether it has a custom plugin

**Example**:
```json
{
  "themes": {
    "peaceful-blue": {
      "type": "imported",
      "source": "~/.hermes/vision/themes/imported/peaceful-blue.hvtheme",
      "installed": "2026-03-14T04:43:48Z",
      "author": "ZenDesigner",
      "title": "Peaceful Blue",
      "description": "Calm blue theme with slow motion",
      "has_plugin": false
    }
  }
}
```

## File Format: .hvtheme

### Structure

```json
{
  "format_version": "1.0",
  "metadata": {
    "name": "theme-name",
    "title": "Theme Title",
    "author": "Creator Name",
    "description": "Theme description",
    "created": "2026-03-14T20:00:00Z",
    "hermes_vision_version": "0.1.1"
  },
  "config": {
    "accent_char": "*",
    "background_density": 0.030,
    "star_drift": 0.10,
    "node_jitter": 0.20,
    "packet_rate": 0.30,
    "packet_speed": [0.04, 0.08],
    "pulse_rate": 0.10,
    "edge_bias": 0.50,
    "cluster_count": 3,
    "palette": ["COLOR_CYAN", "COLOR_BLUE", "COLOR_WHITE", "COLOR_MAGENTA"]
  },
  "plugin": {
    "type": "base" | "custom",
    "code": "base64_encoded_python_if_custom",
    "class_name": "PluginClassName"
  }
}
```

### Two Theme Types

**Config-Only** (Simple):
- Just parameter adjustments
- Uses base ThemePlugin
- Safe to share
- No code execution
- Auto-imports without confirmation

**Custom Plugin** (Advanced):
- Includes Python code
- Custom animations and behaviors
- Base64 encoded
- Requires user confirmation
- Shows code before installing

### Workflow 1: Export Existing Theme

```bash

$ file neural-sky.hvtheme
neural-sky.hvtheme: JSON data
```

### Workflow 2: Preview Before Import

```bash
$ hermes-vision --import theme.hvtheme --preview
Theme: Peaceful Blue
Name: peaceful-blue
Author: ZenDesigner
Description: Calm blue theme with slow gentle motion
Created: 2026-03-14T20:00:00Z
Custom Plugin: No

Use --import without --preview to install
```

### Workflow 3: Import Theme

**Config-Only Theme (No Confirmation)**:
```bash
$ hermes-vision --import peaceful-blue.hvtheme

  Saved to: ~/.hermes/vision/themes/imported/peaceful-blue.hvtheme

Test it with: hermes-vision --theme peaceful-blue
```

**Custom Plugin Theme (With Confirmation)**:

## Security Features

1. **Version Checking**: Reject incompatible formats
2. **Custom Plugin Warning**: Clear warning before executing custom code
3. **Code Review Option**: Shows plugin code before installing
4. **User Confirmation**: Requires explicit approval for custom plugins
5. **Trust Flag**: Power users can skip confirmation with --trust

## Known Limitations

1. **Plugin Dependencies**: Custom plugins must only use:
   - `hermes_vision.plugin.ThemePlugin`
   - `hermes_vision.scene.Particle`
   - `math` module
   - Python stdlib
   
2. **No Sandboxing**: Custom plugin code runs with full privileges
   - Future: Consider sandboxing or code signing

3. **Single Format Version**: Only v1.0 currently supported
   - Future versions will maintain backward compatibility
